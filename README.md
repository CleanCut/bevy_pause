# bevy_pause

A bevy plugin that adds simple pause functionality to a game.

# Example

See the [pause example](./examples/pause.rs) in the `examples` subdirectory, or clone this repository and run `cargo run -r --example pause`.

# Compatibility

|bevy|bevy_pause
|---|---|
| `0.14.*` | `0.1.*` |
| `< 0.14` | Unsupported |
